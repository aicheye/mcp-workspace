# Claude Desktop setup

## 1) Start the HTTP MCP server (recommended in a terminal)
```bash
cd d2l-mcp
npm run mcp:launch
```

## 2) Configure Claude Desktop

Open Claude Desktop config and merge the snippet from:

setup/claude_desktop.snippet.json

Replace:

<ABSOLUTE_PATH_TO_YOUR_REPO> with your actual repo path

Then restart Claude Desktop.
